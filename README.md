# b\_mat\_1013\_numerik\_portfolio
## setup (linux)
vom root des projektes:
```bash
python3 -m venv .b_mat_1013_numerik_portfolio
source .b_mat_1013_numerik_portfolio/bin/activate
pip install -r requirements.txt
python -m ipykernel install --user \
       --name="b_mat_1013_numerik_portfolio-kernel" \
       --display-name="b_mat_1013_numerik_portfolio"
```
